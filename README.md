## Project description:
#### Mind Games Project - five brain training games (Even Check, Calculator, GCD, Arithmetic Progression, Prime Number)

### Minimum requirements:
```
Node.js 13+
```

### Installation:
```
git clone git@github.com:vanya-romashkevich/frontend-project-lvl1.git
make install 
make lint 
```
